## 簡介

TREM-Config-Plugin 適用於 TREM-Lite 3.1.0-rc.1 以上版本。一個讀取 yml 檔案的擴充。

### 安裝方式

- 首先至 [TREM 擴充](https://exptechtw.github.io/trem-plugins/) 或從 TREM-Lite > 設定 > 擴充 > 擴充商店內進行下載
- 下載完成後，重新開啟設定頁面
- 在擴充列表的 config 進行勾選，關閉設定頁面，即可使用
